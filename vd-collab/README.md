# Trendlife UI Mockup — VD 協作說明

歡迎！這份說明會帶你完成整個協作流程，不需要懂 GitHub 或指令列。

---

## 你會需要的

- Mac 電腦
- GitHub 帳號（已有）
- 接受 Karen 寄來的 GitHub 邀請信（才能推送修改）

---

## 第一次設定（只需要做一次）

1. 在 Finder 找到這個資料夾：`vd-collab/`
2. **雙擊** `setup.command`
3. 如果跳出安全性警告，到「系統設定 → 隱私權與安全性」點「仍要開啟」
4. 依照畫面指示：
   - 選擇你要修改的畫面（1 = Mei's view / 2 = Yuki's view）
   - 瀏覽器會自動開啟預覽
   - Finder 會自動開啟工作資料夾

---

## 每次工作流程

```
編輯 HTML 檔案
      ↓
用瀏覽器預覽效果（直接重新整理）
      ↓
改完後雙擊 submit.command
      ↓
輸入這次改了什麼（中文可以）
      ↓
瀏覽器自動開啟 → 按「Create pull request」
      ↓
Karen 收到通知，review 後合併 ✓
```

---

## 修改哪些檔案

你只需要編輯 `constellation/` 資料夾裡的 HTML 檔案：

| 畫面 | 檔案 |
|------|------|
| Mei's view（父母端） | `constellation/mockup-eva-view.html` |
| Yuki's view（孩子端） | `constellation/mockup-yuki-view.html` |

用文字編輯器開啟即可（推薦 VS Code）。
瀏覽器預覽：直接雙擊 HTML 檔案，或把它拖進瀏覽器視窗。

---

## 要換到另一個 view

改完一個 view、送出之後，如果要換到另一個：

1. 再次雙擊 `setup.command`
2. 選擇另一個畫面的號碼
