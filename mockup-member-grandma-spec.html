Design Principles
1
Proactive AI — The butler speaks first. Health reports, schedule nudges, and medication reminders surface before the user has to ask.
2
Persistent quick-actions — The bottom dock is always visible on the home screen, making My Health and Contact Family available in one tap.
3
Senior-adaptive layouts — Font sizes and layouts are adapted based on the senior member's profile and accessibility preferences.
4
Guided interactions — Input and navigation are designed to lead seniors step by step, reducing the technology gap.
5
Family as context — Every health action has a family dimension: share with Eva, notify on appointments, family album curated by AI.

Interaction Patterns
Textile Cards
Bold gradient + repeating-linear-gradient woven texture overlay (::before) + decorative ring motif (::after). Used for Contact Family grid and Quick Actions drawer.
Initials Avatars
All person representations use CSS gradient circles with letter initials. Each family member's color is consistent across all screens (Eva = purple, Wei = navy, Yvonne = emerald, Ethan = red).
Visibility Drawer
Full-screen slide-up overlay using visibility: hidden → visible with transition-delay for animation sequencing. Panel uses transform: translateY(100% → 0).
Persistent Dock
Always visible on Screen 0. Contains expand chevron, two pre-prompt chips, and the chat input bar. Hidden via display: none in goTo() when navigating to sub-screens.
CSS Photo Simulation
Photo grid cells use multi-stop linear-gradient backgrounds to suggest warm, cool, or outdoor scene tones — replacing all emoji for a more realistic photo feel.
