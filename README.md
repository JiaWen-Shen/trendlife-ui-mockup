# TrendLife UI Mockup

互動式手機 UI Mockup，用於 TrendLife AI 管家 Demo Story Line 展示。

## 使用方式

直接用瀏覽器開啟 HTML 檔案，支援手機和桌面瀏覽器。

## 場景覆蓋

| 檔案 | 場景 | 說明 |
|------|------|------|
| `mockup-scene1.html` | 場景 1 | 奶奶的健康檢查 + 防詐守護 |

## 功能

- Eva 首頁：家人狀態列（IG 對話泡泡）、待辦事項（日曆 + tab 切換）、通知中心
- 健康報告詳情：分析摘要、Security Vault 資料流、門診預約
- 防詐守護：攔截畫面、Eva 通知、事件記錄
- 新增提醒：表單、家人選擇、提醒方式
