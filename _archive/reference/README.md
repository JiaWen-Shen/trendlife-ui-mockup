# TrendLife Logo Assets

## Files

| 檔案 | 說明 |
|------|------|
| `logo-mark-red.jpg` | 原始 logo 來源圖（含背景） |
| `logo-full-transparent.png` | 完整 logo（mark + TrendLife 文字），去背 PNG，用於淺色背景 |
| `logo-mark-only.png` | 只有 mark 符號，去背 PNG，用於深色/彩色背景的 avatar 圖示 |
| `logo-reference.html` | Logo 使用規範參考頁，含背景變化、尺寸比例、色票、Do & Don't |

## 使用方式

- **Header / 淺色背景**：使用 `logo-full-transparent.png`
- **深色背景**：使用 `logo-full-transparent.png` + CSS `filter: brightness(0) invert(1)`
- **圓形 avatar 圖示**：使用 `logo-mark-only.png` + CSS `filter: brightness(0) invert(1)`
