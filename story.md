# TrendLife Kaleidoscope — Demo 故事線

序幕：數位時代的孤島 (0:00 - 1:30)
視覺場景：
情境描述：一張電影感的靜態劇照。一家人雖然在溫暖的客廳，但每個人都低頭看著螢幕 。Mei 眉頭深鎖看著工作郵件，孩子們戴著耳機，這代表了科技連結了世界，卻靜默地拆散了家庭 。

核心痛點：家長面臨 AI 時代的兩難：既怕孩子落後，又怕 AI 成為孩子思考的替代品。

第一幕：Creative Folder — 培育批判性思考的種子 (1:30 - 5:30)

場景：放學後的圖書館。Yuki 與同學正在一起看平板。
視覺設計：Creative Folder (創意資料夾) 
 
Yuki 的平板介面。不再是混亂的檔案，而是一個個標註了 作者比例 (Authorship Ratio) 的資料夾（例如：Human 60% / AI 40%） 。

這不是統計數據，而是家長眼中的「成長指標」，也是孩子引以為傲的「原創證明」 。

故事發展：
同學：「這個好酷！妳真幸運，能使用這種付費 GenAI。」
Yuki：「因為我有一個很 Cool 的媽媽，她選了這個超棒的產品——Kaleidoscope。它讓我變得更強更有創造力。」
UI 精度 (Agentic AI 的體現)：
啟發式對話：點開 Yuki 的創作對話日誌。AI 不會直接給答案，而是跳出視窗詢問：「這段音樂你想要什麼情緒？」或「你覺得這張圖的色調該暖一點還是冷一點？」 。

這展示了孩子的 品味 與 批判性思考，體現了家長設定的教育價值觀 Agent 。

第二幕：Family Context View — 危機中的隱形導航 (5:30 - 10:30)

場景：Mei 在辦公室開會，手機不斷跳出訊息風暴。
情境描述：
今天是奶奶的慶生晚餐，奶奶正熱火朝天地煮著赤飯。
媽媽Mei的訊息風暴：Hiro 練習延後 ；爸爸 Kenji 塞車中，卻還沒去拿藥 。

突發危機：奶奶的穿戴裝置發出 「血壓過高 (High BP)」 警示 ，媽媽Mei的手機也第一時間得到消息。

Agentic AI 的智慧救援 (Family Context View) ：

Agent 介入：AI 語音告知奶奶：「奶奶，我們現在休息一下吧。Kenji 10 分鐘後就到。」
跨成員協調：系統自動連結爸爸的交通 Agent 與奶奶的藥局處方箋任務，並修改導航路線 。

情感激勵：AI 對 Mei 發出提醒：「已提醒奶奶身體不適休息中，今天是她的生日，也已按您預定準備好生日蛋糕了。」 。

收尾畫面：慶生晚餐大圖。燈光美、氣氛佳，全家人圍繞著奶奶開心慶生，危機轉化為溫馨的連結。

第三幕：深夜客廳的「family agent報告」 (10:30 - 14:00)
場景：深夜客廳。Mei 與老公放鬆地坐在沙發上，Mei 點開「本日管家報告」。
視覺呈現：本日家庭總結 (Daily family Report)
Security Activities (防護報告)：
以圓餅圖顯示 24 小時內的全家防護紀錄（如：攔截 15 通詐騙電話、8 個危險連結） 。

點開 Scam Call 列表，揭露系統在背景默默攔截了試圖騷擾奶奶的醫療詐騙電話 。

Kids Learning (成長報告) ：
點開 Yuki 的頭像。展示今日的 「創意決策鏈」 與原創比例提升趨勢 。

媽媽Mei 看到 Yuki 堅持自己的風格、推翻 AI 建議的過程，感到由衷欣慰。
產業變革點：這就是 Agentic AI 的體系結構：每個家人都擁有專屬的Agent，在嚴格守護個人隱私與數位邊界的同時，這些 Agent 共同遵循著一個核心的大規範——『技術必須服務於家庭的連結，而非取代成員的獨立。』  只有在『家庭協作』的關鍵時刻，Agent 才會根據家長設定的價值哲學 ，在尊重每個人主體性的前提下，安全地交換必要資訊 。這確保了科技在提供便利的同時，依然守護著家庭中最重要的信任與成長空間。
